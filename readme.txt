Оплата через Платрон в модуле JigoShop

== Description ==

С помощью Платрон вы сможете принимать платежи более чем через 20 платежных систем, объединенных единым техническим шлюзом и коммерческими условиями подключения.

== Installation ==

1. Загрузите плагин
2. Устанавливаем модуль (Плагины->Установить и Активировать)
3. Регистрируемся на platron.ru
4. Заходим JogoShop->Расчет заполняем требуемые данные *

* Если не хотите принимать заказ к оплате - поменяйте статус заказа на отменен или придержан.
